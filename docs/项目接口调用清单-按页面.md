## 1. 全局请求规则

- 基础地址：`http://localhost:3000`（前端 `settings.baseUrl`）。
- 鉴权方式：需要登录的接口都要带 `Authorization: Bearer <token>`。
- 统一响应判定：前端以 `code` 判成功，`200`/`202` 视为成功。
- 生图接口是异步受理：创建接口返回 `taskId/queryPath/ssePath`，最终结果要查任务状态。

### POST `/app/banana-CreateImage`
- 鉴权：需要 JWT。
- 请求参数：
```json
{
  "type": "text-to-image",
  "model": "gemini-2.5-flash-image-preview",
  "prompt": "A cat",
  "aspectRatio": "1:1",
  "imageSize": "1K",
  "outputFormat": "jpg",
  "skuId": 10,
  "skuCode": "banana_1k",
  "imageUrls": ["/appTempData/2026/04/xx.png"],
  "imageUrl": "/appTempData/2026/04/xx.png",
  "idempotencyKey": "uuid"
}
```
- 参数约束：
- `model`、`prompt` 必填。
- `type=image-to-image` 时 `imageUrls` 或 `imageUrl` 必填。
- `skuId` 若传必须为正整数。
- 成功返回（`code: 202`，异步受理）：
```json
{
  "code": 202,
  "message": "任务已受理",
  "data": {
    "taskId": "uuid",
    "status": "pending",
    "queryPath": "/app/banana-CreateImage/tasks/uuid",
    "ssePath": "/app/banana-CreateImage/tasks/uuid/stream",

      "skuId": 10,
      "skuCode": "banana_1k",
      "unitPoints": 20,
      "quantity": 1,
      "totalPoints": 20,
      "remainingPoints": 980,
      "imageSize": "1K"
    }
  }
}
```

### POST `/app/image-to-image`
- 鉴权：需要 JWT。
- 请求参数（修正后，`model` 必填，不是 `modelValue`）：
```json
{
  "prompt": "A cat with hat",
  "size": "1024x1024",
  "aspectRatio": "1:1",
  "model": "gpt-image-2",
  "billingModelKey": "gpt-image-2",
  "skuCode": "gpt2_1k",
  "imageUrl": ["/appTempData/2026/04/xx.png"],
  "uploadToCos": true,

```
- 参数约束：
- `prompt`、`imageUrl[]`、`model` 必填。
- `uploadToCos` 仅支持 `true`。
- 成功返回：与 `/app/text-to-image` 同结构（`code: 202`，通过 `data.upload.queryPath` 查询结果）。

### GET `queryPath`（动态任务查询）
- 可能路径：
- `/app/banana-CreateImage/tasks/{taskId}`
- `/app/text-to-image/tasks/{taskId}`
- 请求参数：路径参数 `taskId`。
- 成功返回（`code: 200`）：
```json
{
  "code": 200,
  "data": {
    "taskId": "uuid",
    "status": "running",
    "type": "text-to-image",
    "provider": "gpt",
    "model": "gpt-image-2",
    "skuCode": "gpt2_1k",
    "imageSize": null,
    "outputFormat": "jpg",
    "prompt": "A cat",
    "aspectRatio": "1:1",
    "size": "1024x1024",
    "quality": "medium",
    "style": "vivid",
    "quantity": 1,
    "imageUrls": [],
    "attempts": 1,
    "maxAttempts": 4,
    "nextRetryAt": null,
    "errorCode": null,
    "errorMessage": null,
    "recordId": 999,
    "cosUrl": "/generated/xx.jpg",
    "previewUrl": "/generated/xx.jpg?imageMogr2/thumbnail/1024x>/strip/quality/75/format/webp",
    "thumbnailUrl": null,
    "createdAt": 1745913600000,
    "updatedAt": 1745913605000,
    "queueWaitMs": 120,
    "workerDurationMs": null,
    "traceId": "uuid"
  }
}
```
- 终态判断：`status=success` 或 `status=failed`。

### GET `ssePath`（动态 SSE）
- 示例路径：`/app/banana-CreateImage/tasks/{taskId}/stream`
- 鉴权：需要 JWT。
- 事件类型：`task_status`、`task_success`、`task_failed`、`ping`。
- 事件 `data` 字段结构与任务查询 `data` 基本一致。
